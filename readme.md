#### Installation

```bash
wget https://raw.githubusercontent.com/NRGetRekt/easyinstall/main/easyinstall && bash easyinstall
```

##### Usage

To Start the service use: 
```bash
badvpn start
```

To Stop the service use: `badvpn stop`

To Uninstall Badvpn use: `badvpn uninstall`
