#### Installation

`wget https://raw.githubusercontent.com/NRGetRekt/master/easyinstall && bash easyinstall`

##### Usage

To Start the service use: `badvpn start`

To Stop the service use: `badvpn stop`

To Uninstall Badvpn use: `badvpn uninstall`
